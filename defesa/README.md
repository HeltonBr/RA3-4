# Kit de Defesa - Fase 3

Arquivos de apoio para demonstrar o analisador semantico:

- `demo_valido_base.txt`
- `demo_valido_alterado.txt`
- `demo_erro_lexico.txt`
- `demo_erro_sintatico.txt`

Comando:

```powershell
python AnalisadorSemantico.py defesa/demo_valido_base.txt
```

Para erros semanticos, usar os casos em `tests/invalidos/semantico_*.txt`.
