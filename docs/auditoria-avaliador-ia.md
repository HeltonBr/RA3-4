# Auditoria do Padrao de Avaliacao por IA

Medidas aplicadas para reduzir risco na avaliacao automatizada:

- README objetivo com comandos reais e artefatos listados.
- Documentacao separada para gramatica atribuida, regras de tipos, tabela de simbolos, arvore atribuida e erros.
- Decisoes explicitas para cada inconformidade do dossie.
- Testes automatizados que conferem requisitos de arquivos validos e invalidos.
- Mensagens de erro com tipo, linha, coluna e causa.
- `generated/` sempre atualizado por uma execucao valida final.
- Compatibilidade com tokens serializados antigos sem abandonar a entrada por fonte bruto.

Risco residual: o nome exato do grupo Canvas ainda deve ser confirmado antes da publicacao do repositorio.
